Conheça o Jackknife

Um exemplo de método de reamostragem.

O que é?

Em estatística, o nome jacknife foi usado por Tukey (1958) para descrever uma perspectiva geral para realizar testes de hipóteses e calcular intervalos de confiança. O Jackknife consiste em recalcular um particular estimador, eliminando sistematicamente cada observação individual (leave-one-out)!

Como funciona?

Por exemplo, uma amostra de tamanho \(n=4\) pode ser representada da seguinte forma:

Figura: Representação do Jackknife em cores.

  • Jack 1 deixa apenas a observação amarela de fora,

  • Jack 2 retoma a observação excluída anteriormente (amarela) e deixa de fora a observação verde, assim por diante.

  • As cores das caselas representam observações que na prática são mensurações de um particular evento (números), que inclusive podem coincidir.

  • Deixa-se de fora um por vez até que se obtenha n amotras, cada uma composta pelas observações da amostra original exceto a j-ésima observação.

Um pouco de Estatística

Conceitos fundamentais de Estatística são necessários para compreender o funcionamento de métodos de estimação por reamostragem, afinal todo estimador é necessariamente uma estatítica do qual extrairemos inferências sobre uma quantidade de interesse (o parâmetro)!

Em poucas palavras, qualquer função da amostra que não depende do parâmetro é uma estatística, por exemplo:

\[\hat\theta(X_1, X_2, \dots, X_n) = \frac{1}{n}\sum\limits_{i=1}^{n}X_i = \bar{X}.\]

pode ser o estimador da média populacional \(\theta = \mu\), quando um estimador é aplicado a uma amostra proveniente da população em estudo obtemos um valor real \(t\), chamado estimativa.

Por exemplo, considere uma amostra uma amostra \(\mathbf{x}\) de tamanho \(n = 4\) \(\mathbf{x}=(x_1,x_2,x_3, x_4)=(4,1,2,3)\) e o estimador \(\bar{X}\) para o parâmetro \(\theta\), ou seja, \(\hat\theta = \bar{X}\). A estimativa pontual é obtida quando aplicamos a estatística referente ao estimador aos dados provenientes de uma amostra da população de interesse, nesse caso, a estimativa pontual é dada por \(t = \frac{1}{4}\sum\limits_{i=1}^{4}x_i = \frac{4+1+2+3}{4}=2,5.\)

Como fazer Inferência baseada no método Jackknife?

No contexto de Inferência Estatística, além das estimavas pontuais, ainda é possível obter estimativas intervalares e realizar testes de hipóteses com base em amostras observadas da população alvo (Veja Morettin (2017)). Nesta breve introdução será apresentada a estimativa pontual Jackknife e sua estimativa intervalar correspondente, em que não há qualquer suposição sobre a distribuição dos dados em estudo.

1- Estimativa Parcial

Dada uma amostra \((x_1,x_2,\dots ,x_n)\) qualquer e um estimador para o parâmetro de interesse, a estimativa parcial consiste em simplesmente obter estimativas pontuais \(t_{-j}\) para cada amostra Jack!

2- Pseudo-valores

Os pseudo-valores \(t^∗_j\) farão parte do cáculo da estimativa pontual Jack abaixo. \[t^∗_j = nt-(n-1)t_{-j}\] Note que \(t\) é a estimativa calculada usando a amostra original (sem retirada) e que \(n\) é o tamanho da amostra.

3 - Estimativa Pontual Jackknife

A estimativa pontual Jack é a média aritmética dos pseudo-valores.

\[t^*=\frac{1}{n}\sum\limits_{j=1}^{n}t^∗_j \] Logo, o cálculo da estimativa Jack consiste apenas em computar média aritmética dos pseudo-valores obtidos com as amostras Jackknife.

Só isso? Veja o exemplo

Exemplo 2.1, Manly (2006): Suponha que seja extraída uma amostra aleatória de tamanho 20 de uma certa população na qual o desvio-padrão populacional \(\sigma\) é o parâmeto de interesse do estudo. Obtenha uma estimativa intervalar para \(\sigma\) a partir de \(\hat\sigma=\frac{1}{n}\sum\limits_{i=1}^n (X_i−\bar{X})^2\).

\[ 3,56;0,69;0,10;1,84;3,93;1,25;0,18;1,13;0,27;0,50;0,67;0,01;0,61;0,82;1,70;0,39;0,11;1,20;1,21;0,72. \]

Obs.:

  • Não é recomendável fazer suposições sobre a distribuição da amostra ou do estimador para uma amostra de tamanho 20.

  • Além disso, não se sabe qual é a distribuição da população da qual provém a amostra.

  • No entanto, o método jackknife permite obter estimativas pontuais e intervalares para o parâmetro.

Vamos aplicar essa técnica de reamostragem no R!

Define-se uma função chamada sigma_chapeu() que irá representar o estimador \(\hat\sigma\) dado na questão.

sigma_chapeu <- function(x){ 
  x <- x[!is.na(x)] # remove dados faltantes
  xbarra <- mean(x)
  desvio <- x-mean(x)
  n <- length(x)
  return(sqrt(sum(desvio^2)/n)) 
} 

t_amostra <- sigma_chapeu(amostra); t_amostra
## [1] 1.032848

Para o cálculo das estimativas parciais \((t_{−1},t_{−2},\dots,t_{−n})\), a função apply() é usada abaixo para aplicar sigma_chapeu() em todas as linhas “MARGIN=1” da matrizjack, o uso de funções apply evita loops como for() e while(), o excesso de loops pode tornar a programação computacionalmente ineficiente.

t_par <- apply(X = matrizjack, MARGIN = 1 , FUN = sigma_chapeu)

Por fim, a estimava pontual jackknife é dada pela média aritmética dos pseudo-valores.

estjack <- mean(pseudov)
print(estjack)
## [1] 1.096158

Segundo Manly (2006), a estimativa intervalar aproximada para \(\sigma\) a um nível de \(100(1−\alpha)\%\) de confiança é dada por um intervalo de confiança t-Student:

\[\left[t^*\pm t_{\frac{\alpha}{2},n-1}\times\frac{s}{\sqrt{n}}\right]\]

Ao fixar \(\alpha = 0,05\), temos o intervalo Jackknife de 95% de confiança:

 alfa <- 1-0.95  
 s <- sd(pseudov)

 L <- estjack - qt(1-alfa/2,df = n-1) * s/sqrt(n) 
 U <- estjack + qt(1-alfa/2,df = n-1) * s/sqrt(n) 
 c("L"=L,"U"=U) 
##        L        U 
## 0.525173 1.667142

Conclusão

O termo jackknife (canivete em tradução livre) remete à portabilidade e a possibilidade de uso em diversas situações. Essa técnica, desenvolvida por Maurice Quenouille (1949), é particularmente útil em situações em que a distribuição do estimador é desconhecida. O método Jackknife não é computacionalmente intensivo e é facilmente implementado com conhecimento básico em inferência e um pouco de R! Considerando que a amostra provém de uma população Exponencial(1), obtivemos um intervalo de confiança que contém o real valor do parâmetro \(\sigma = 1\) com uma amostra de tamanho 20, usando um estimador do qual não sabemos a distribuição.
